Let us see how a program looks like in my Blog

  • Sample C Program
int factorial (num) {
    if (num == 0) {
        return 1;
    }
    return num * factorial (num - 1);
}
  • Sample Haskell Program to calculate factorial of a number
factorial :: num -> num
factorial 0 = 1
factorial num = num * factorial (num - 1)
